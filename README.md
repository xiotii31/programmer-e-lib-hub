# Digital Library System

A simple web-based Digital Library System built using HTML, CSS, and JavaScript for browsing and accessing digital books online.

## Features

* View digital books
* Search books easily
* Open PDF files online
* Simple and user-friendly interface
* Free online hosting using GitHub Pages

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```plaintext
digital-library/
│
├── index.html
├── style.css
├── script.js
├── books/
├── images/
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.
